### 系统功能

#### 资产管理

 - 新建设备资产单
 - 新建/修改/删除资产调拨单
 - 发起资产调拨的审批流程
 - 归档资产调拨单

 ![资产管理](https://developer.steedos.com/assets/no-code/asset_printer.png)

#### 设备资产

 - 新建设备资产单

 ![资产管理](https://developer.steedos.com/assets/no-code/assetallocation_6.png)

#### 资产调拨

 - 新建/修改/删除资产调拨单
 - 发起审批

 ![资产管理](https://developer.steedos.com/assets/no-code/asset_workflow_7.png)

#### 资产调拨流程

 - 发起、处理各类资产调拨流程
 - 资产调拨流程与对应资产调拨单的数据同步

### 系统安装

#### 安装 yarn
```
npm i yarn -g
```

#### 国内建议使用npm淘宝镜像
```
npm config set registry http://registry.npm.taobao.org/
```

#### 使用yarn安装依赖包
```
yarn
```

#### 启动服务器
```
yarn start
```
